# Pokemon Random
Um website que consome api e retorna na tela um pokemon por vez.
## 🚀 Tecnologias Usadas
- JSX
- CSS 
- JavaScript 
- ReactJS
## ℹ️ Para baixar todas dependencias
~~~ 
npm install
  ou
yarn install
~~~
